### Docker 명령어

<br>

| 명령어            |  설명                   |
| ----------------- | :--------------------: |
| docker run <이미지 이름>  |                      |
| docker ps         | 현재 실행중인 컨테이너 |
| docker ps -a      |   모든 컨테이너 조회   |
| docker create <이미지 이름> | 이미지 파일 스냅샷 디스크에 올림 |
| docker start -a <시작할 컨테이너 이름/ID> | 컨테이너 프로세스에 올림 |
